# Proyecto Integrador

Proyecto Java Maven de ejemplo ubicado en `demo/`.

## Descripción

Aplicación Java simple construida con Maven. El proyecto usa Java 17 y contiene una clase principal en `src/main/java/com/example/Main.java`.

## Estructura del proyecto

- `demo/pom.xml` - archivo de configuración de Maven.
- `demo/src/main/java/` - código fuente principal.
- `demo/src/test/java/` - pruebas unitarias.
- `demo/target/` - artefactos compilados generados por Maven.

## Requisitos

- Java 17
- Maven 3.x

## Comandos útiles

Desde el directorio raíz del proyecto:

```bash
cd demo
mvn clean compile
mvn test
mvn package
```

## Notas

Este README es una guía básica para compilar y ejecutar el proyecto. Ajusta los comandos según tu entorno si es necesario.
